# VerseFlow — Downloads

Public releases and download site for **VerseFlow**, the live scripture
detection app for church media teams.

- **Website / download page:** https://stephanas-soft.github.io/verseflow-releases/
- **Latest Windows installer (always newest):**
  https://github.com/STEPHANAS-SOFT/verseflow-releases/releases/latest/download/VerseFlow-Setup.exe
- **Update manifest used by the app:**
  https://github.com/STEPHANAS-SOFT/verseflow-releases/releases/latest/download/latest.json

Releases are published automatically by the VerseFlow build pipeline.
